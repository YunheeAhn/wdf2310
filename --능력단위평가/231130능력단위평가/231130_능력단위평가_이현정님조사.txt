*병원사이트를 방문하는 주요 목적
 - 진료서비스(예약, 의료진/진료과에 대한 정보, 병원정보)
 - 건강정보

문제의 원인

1. 복잡한 진료예약 화면
 - 진료예약 부분이 목적별로 나눠져있음에도 불구하고 눈에 잘 들어오지 않아 불편함
=> 항목별로 눈에 잘 들어오도록 나누어 예약하기 쉽게 디자인
2. 예약서비스와 발급서비스가 함께 있어 구분이 쉽지 않음
=> 서비스가 다른 부분은 분류하여 제작
3. 병원에 대한 정보가 미흡함
 - 병원정보(건물외관, 오시는길, 주차안내, 병원생활안내 등)에 대하여 아이콘이 있으나 눈에 잘 들어오지 않음
=> 한 눈에 알아볼 수 있도록 제작
4. 건강정보를 알아보기 어려움
 - 병원사이트를 방문하는 사람은 건강에 대한 염려가 있는 사람들임.
=> 따라서 건강정보를 쉽게 알아볼 수 있도록 메인페이지에 항목 추가
5. 의료진에 대한 정보 부족
 - 해당 병원에 어떤 진료과가 있는지, 어떤 의료진들이 있는지 알기에 어려움
=> 진료과에 대한 정보, 의료진에 대한 정보 추가
6. 상단 진료예약배너가 눈에 들어오지 않고 시선이 분산됨
=> 사이드에 창을 띄워 홈페이지 시선을 분산되지 않고 한 눈에 볼 수 있도록 함
7. 필요한 서비스가 전체적으로 가로형으로 되어 있어 눈에 들어오지 않음
 => 항목별로 구분지어 재배치
