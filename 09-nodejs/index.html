node.js 설치하기

설치 방법

node.js 홈페이지가서 LTS 설치

vscode 터미널에 입력할 것

1. node -v

2. npm -v

3. npm init -y

4. npm install

5. npm install parcel --save-dev

npm run dev : npm install parcel --save-dev 설치 후 개발자용 서버 돌리기

error뜨는 경우, 왼쪽 파일 탭에서 parcel-cache를 지워야 한다

서버종료는 ctrl + c
